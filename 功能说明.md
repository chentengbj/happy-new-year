# 功能修改说明

## 已完成的修改

### 1. ✅ 删除图表分析、时间对比、公司对比功能
- 已删除HTML中的相关视图
- 已删除导航按钮
- 已删除JavaScript中的所有图表相关函数
- 已移除Chart.js依赖

### 2. ✅ 修改领域选项
- **教研岗位** → **教研成果**：收集教研成果相关信息
- **市场营销** → **获客增长**：收集获客增长相关信息
- **新增招聘信息**：收集在线教育类公司相关的招聘信息

### 3. ✅ 招聘信息功能
招聘信息包含以下字段：
- `type`: '招聘'
- `company`: 公司名称
- `title`: 标题（会高亮AI+教育相关内容）
- `jobTitle`: 岗位名称
- `jobDescription`: 职位描述（JD）
- `jobFunction`: 职能分类
- `workExperience`: 工作经验要求
- `salaryRange`: 薪资范围
- `priority`: 优先级
- `categories`: 分类（包含'招聘信息'）
- `date`: 日期
- `source`: 真实来源链接（必须可验证）

**职能分类包括：**
- 产品研发类
- 讲师/教研类
- 市场获客类
- 用户运营类
- 品牌公关类
- 销售类
- 供应链相关类
- 视频动画类
- 原画/视觉设计类

### 4. ✅ AI+教育高亮功能
- 标题中自动识别并高亮AI+教育相关关键词
- 关键词包括：AI、人工智能、AI+、智能、机器学习、算法、AI教育、AI+教育
- 高亮样式：黄色背景，深色文字，加粗显示

### 5. ✅ 信息验证机制
- 所有信息必须有真实来源链接
- 自动验证链接有效性
- 排除示例链接（example.com）
- 无效来源的信息会被过滤
- 所有信息可通过"查看原文"按钮验证

## 数据格式

### 普通新闻格式
```javascript
{
    type: '新闻',
    company: '公司名',
    title: '标题（AI+教育相关内容会自动高亮）',
    content: '内容',
    priority: '重点' 或 '次重点',
    categories: ['分类1', '分类2'],
    date: 'YYYY-MM-DD',
    source: '真实来源链接'
}
```

### 招聘信息格式
```javascript
{
    type: '招聘',
    company: '公司名',
    title: '标题（AI+教育相关内容会自动高亮）',
    jobTitle: '岗位名称',
    jobDescription: '职位描述',
    jobFunction: '职能分类',
    workExperience: '工作经验要求',
    salaryRange: '薪资范围',
    priority: '重点' 或 '次重点',
    categories: ['招聘信息', '其他分类'],
    date: 'YYYY-MM-DD',
    source: '真实来源链接（招聘网站链接）'
}
```

## 使用说明

1. **筛选功能**
   - 可按重要性、公司、领域筛选
   - 招聘信息可按职能分类筛选
   - 支持关键词搜索

2. **查看原文**
   - 所有信息都有"查看原文"按钮
   - 点击可在新标签页打开真实来源
   - 无效来源会显示"来源待验证"

3. **AI+教育高亮**
   - 标题中包含AI+教育相关关键词会自动高亮
   - 便于快速识别AI+教育相关信息

4. **招聘信息显示**
   - 招聘信息有特殊的卡片样式
   - 显示岗位名称、职能、经验、薪资等详细信息
   - 可按职能分类筛选

## 数据收集脚本

运行 `collect_news.py` 脚本时：
- 自动验证所有来源链接
- 自动识别AI+教育相关内容
- 自动分类招聘信息
- 过滤无效来源的信息

## 注意事项

1. **数据真实性**：所有数据必须来自真实来源，禁止编造
2. **链接验证**：所有链接必须可访问，系统会自动验证
3. **AI+教育重点**：重点关注AI+教育相关的新闻和招聘信息
4. **招聘信息完整性**：招聘信息应包含岗位名称、JD、经验、薪资等完整信息
