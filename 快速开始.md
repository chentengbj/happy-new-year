# 快速开始指南

## 第一步：安装依赖

打开终端，进入 `code` 目录，运行：

```bash
cd "/Users/chenteng/Desktop/cursor 启动/code"
pip3 install -r requirements.txt
```

如果遇到权限问题，使用：
```bash
pip3 install --user -r requirements.txt
```

## 第二步：测试运行

运行数据收集脚本：

```bash
python3 collect_news.py
```

脚本会自动：
1. 收集教培行业资讯
2. 识别公司、分类、优先级
3. 更新 `data.js` 文件

## 第三步：设置自动运行（可选）

### macOS/Linux 用户

运行设置脚本：
```bash
./setup_cron.sh
```

或者手动设置：
```bash
crontab -e
```

添加以下行（每周一上午9点运行）：
```
0 9 * * 1 cd /Users/chenteng/Desktop/cursor\ 启动/code && /usr/bin/python3 collect_news.py >> /Users/chenteng/Desktop/cursor\ 启动/code/collect_log.txt 2>&1
```

### Windows 用户

1. 打开"任务计划程序"
2. 创建基本任务
3. 设置每周运行一次
4. 程序路径：`python` 或 `python3`
5. 参数：`collect_news.py`
6. 起始位置：`code` 目录路径

## 注意事项

1. **首次运行**：可能需要一些时间收集数据
2. **数据源**：脚本中的示例数据源可能需要根据实际情况调整
3. **网络连接**：确保网络连接正常
4. **数据质量**：自动收集的数据建议人工审核

## 手动添加数据

如果自动收集的数据不够，可以手动编辑 `data.js` 文件添加数据。

## 查看日志

如果设置了定时任务，日志文件在：`collect_log.txt`
