## 如何获取具体链接

### 新闻链接

2. **搜索相关公司**的新闻
3. **点击具体文章**，复制文章URL
4. **更新到data.js**中

### 招聘链接
1. **访问招聘网站**（BOSS直聘、猎聘网、拉勾网等）
2. **搜索相关公司**的岗位
3. **点击具体岗位**，复制岗位详情页URL
4. **更新到data.js**中

## 链接验证方法

### 方法1：使用验证脚本
```bash
python3 verify_links.py
```
（需要安装requests库：`pip3 install requests`）

### 方法2：手动验证
1. 在浏览器中打开每个链接
2. 检查是否能正常访问
3. 检查是否能看到相关内容
4. 如果无效，更新为有效链接

### 方法3：批量更新
```bash
python3 fix_invalid_links.py
```
（会自动替换无效链接为有效链接）

## 招聘信息收集要求

### 必须包含的字段
- `type`: '招聘'
- `company`: 公司名称
- `title`: 标题（包含公司名和岗位名）
- `jobTitle`: 岗位名称
- `jobDescription`: 职位描述（JD）
- `jobFunction`: 职能分类
- `workExperience`: 工作经验要求
- `salaryRange`: 薪资范围
- `source`: **真实招聘网站链接**（必须可验证）

### 数据源优先级
1. **BOSS直聘** - 主要数据源
2. **猎聘网** - 补充数据源
3. **拉勾网** - 补充数据源
4. **公司官网** - 备用数据源

### 职能分类
必须使用以下9个分类之一：
- 产品研发类
- 讲师/教研类
- 市场获客类
- 用户运营类
- 品牌公关类
- 销售类
- 供应链相关类
- 视频动画类
- 原画/视觉设计类
