# 安装和运行说明

## 系统要求

运行数据收集脚本需要：
1. Python 3.6 或更高版本
2. Xcode 命令行工具（macOS）

## 安装步骤

### 第一步：安装 Xcode 命令行工具（macOS）

如果运行脚本时出现 "No developer tools were found" 错误，需要先安装 Xcode 命令行工具：

**方法1：自动安装（推荐）**
```bash
xcode-select --install
```
然后按照提示完成安装。

**方法2：手动安装**
1. 打开 App Store
2. 搜索并安装 Xcode
3. 打开 Xcode，接受许可协议

### 第二步：安装 Python 依赖

安装完 Xcode 命令行工具后，运行：

```bash
cd "/Users/chenteng/Desktop/cursor 启动/code"
pip3 install -r requirements.txt
```

如果遇到权限问题：
```bash
pip3 install --user -r requirements.txt
```

### 第三步：测试运行

**测试模式（不依赖网络）：**
```bash
python3 test_collect.py
```

**完整模式（需要网络连接）：**
```bash
python3 collect_news.py
```

## 常见问题

### 1. "No developer tools were found"
**解决方案：** 运行 `xcode-select --install` 安装 Xcode 命令行工具

### 2. "pip3: command not found"
**解决方案：** 
- macOS: 安装 Xcode 命令行工具
- 或使用 `python3 -m pip` 代替 `pip3`

### 3. "ModuleNotFoundError"
**解决方案：** 确保已安装所有依赖：
```bash
pip3 install -r requirements.txt
```

### 4. 网络连接问题
**解决方案：** 
- 检查网络连接
- 某些网站可能需要 VPN
- 可以先用测试模式验证脚本逻辑

## 替代方案

如果无法安装 Python 环境，可以：

1. **手动更新数据**
   - 直接编辑 `data.js` 文件
   - 按照现有格式添加新数据

2. **使用在线 Python 环境**
   - 使用 Google Colab
   - 使用 Replit
   - 将脚本上传到这些平台运行

3. **使用其他语言重写**
   - Node.js 版本
   - JavaScript 版本（浏览器中运行）

```bash

# 检查 pip

```
