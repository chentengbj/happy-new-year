# 快速启动指南 - 局域网分享

## 🚀 方法1：使用启动脚本（最简单）

### 步骤：

1. **打开终端（Terminal）**

2. **进入项目目录**
   ```bash
   cd "/Users/chenteng/Desktop/cursor 启动/code"
   ```

3. **运行启动脚本**
   ```bash
   ./start_server.sh
   ```

4. **查看输出信息**
   脚本会显示：
   ```
   🚀 启动本地服务器...
   📡 本机访问: http://localhost:8000
   🌐 局域网访问: http://192.168.1.100:8000
   ```

5. **访问网站**
   - **本机访问**：在浏览器打开 `http://localhost:8000`
   - **局域网访问**：告诉其他人访问 `http://你的IP:8000`（例如：`http://192.168.1.100:8000`）

6. **停止服务器**
   按 `Ctrl + C` 停止服务器

---

## 🔧 方法2：手动启动（如果脚本不工作）

### 步骤：

1. **打开终端**

2. **进入项目目录**
   ```bash
   cd "/Users/chenteng/Desktop/cursor 启动/code"
   ```

3. **启动Python服务器**
   ```bash
   python3 -m http.server 8000 --bind 0.0.0.0
   ```

4. **查看本机IP地址**（告诉其他人）
   ```bash
   # 方法1：查看IP
   ipconfig getifaddr en0
   
   # 方法2：查看所有IP
   ifconfig | grep "inet " | grep -v 127.0.0.1
   ```

5. **访问网站**
   - 本机：`http://localhost:8000`
   - 局域网：`http://你的IP:8000`

## 📱 让手机/平板访问

### 前提条件：
- 手机/平板和电脑在**同一个WiFi网络**下

### 步骤：

1. **启动服务器**（使用上面的方法）

2. **查看电脑的IP地址**
   ```bash
   ipconfig getifaddr en0
   ```
   假设显示：`192.168.1.100`

3. **在手机浏览器输入**
   ```
   http://192.168.1.100:8000
   ```

## 🔍 常见问题

### Q1: 提示 "Permission denied" 或 "权限被拒绝"
**解决方案：**
```bash
chmod +x start_server.sh
./start_server.sh
```

### Q2: 提示 "python3: command not found"
**解决方案：**
- macOS 通常自带 Python 3，如果没有，安装：
  ```bash
  xcode-select --install
  ```
- 或者使用完整路径：
  ```bash
  /usr/bin/python3 -m http.server 8000 --bind 0.0.0.0
  ```

### Q3: 其他人无法访问
**检查清单：**
1. ✅ 确认电脑和访问设备在**同一WiFi网络**
2. ✅ 确认防火墙没有阻止端口8000
3. ✅ 确认IP地址正确
4. ✅ 确认服务器正在运行（终端没有关闭）

### Q4: 如何查看当前IP地址？
```bash
# macOS
ipconfig getifaddr en0

# 或者查看所有网络接口
ifconfig | grep "inet " | grep -v 127.0.0.1
```

### Q5: 如何更改端口号？
编辑 `start_server.sh` 文件，将 `8000` 改为其他端口（如 `8080`、`3000` 等）

## 💡 使用技巧

### 技巧1：后台运行服务器
如果想关闭终端窗口但保持服务器运行：
```bash
nohup python3 -m http.server 8000 --bind 0.0.0.0 > server.log 2>&1 &
```

停止后台服务器：
```bash
# 查找进程
ps aux | grep "http.server"

# 停止进程（替换PID为实际进程号）
kill PID
```

### 技巧2：使用其他端口
如果8000端口被占用，可以使用其他端口：
```bash
python3 -m http.server 8080 --bind 0.0.0.0
```
然后访问：`http://localhost:8080` 或 `http://你的IP:8080`

### 技巧3：查看访问日志
服务器运行时会显示访问日志，可以看到谁在访问网站。
