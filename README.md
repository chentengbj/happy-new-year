# 教培行业动向追踪系统

## 🌐 访问网站

### 本地访问（当前状态）
目前网站是本地文件，只能在本机访问：
- 直接打开 `index.html` 文件
- 或使用本地服务器（见下方）

### 让其他人访问

**快速方式 - 本地服务器（局域网）：**
```bash
# 运行启动脚本
./start_server.sh

# 或手动启动
python3 -m http.server 8000 --bind 0.0.0.0
```

然后告诉其他人访问：`http://你的IP地址:8000`

**公开访问 - 部署到 GitHub Pages（免费，推荐）：**
详见 [GitHub部署指南.md](./GitHub部署指南.md)

**其他部署方式：**
详见 [部署指南.md](./部署指南.md)


自动收集和展示互联网教培行业的动向信息，重点关注好未来、希望学、学而思、新东方、作业帮、高途等公司。

## 功能特性

- 📊 **多视图展示**：列表视图、图表分析、时间对比、公司对比
- 🔍 **智能筛选**：按公司、领域、优先级、标签筛选
- 📈 **数据可视化**：多种图表展示数据趋势
- ⭐ **收藏功能**：收藏重要资讯
- 🏷️ **标签管理**：自定义标签分类
- 📥 **数据导出**：导出JSON格式数据
- 📱 **移动端适配**：响应式设计

## 安装和使用

### 1. 安装依赖

```bash
cd code
pip install -r requirements.txt
```

### 2. 运行数据收集脚本

#### 方式1：手动运行
```bash
python collect_news.py
```

#### 方式2：设置定时任务（每周自动运行）

**macOS/Linux (使用crontab):**
```bash
# 编辑crontab
crontab -e

# 添加以下行（每周一上午9点运行）
0 9 * * 1 cd /Users/chenteng/Desktop/cursor\ 启动/code && /usr/bin/python3 collect_news.py
```

**Windows (使用任务计划程序):**
1. 打开"任务计划程序"
2. 创建基本任务
3. 触发器：每周一
4. 操作：启动程序 `python collect_news.py`
5. 起始于：`code` 目录路径

### 3. 打开网页

直接在浏览器中打开 `index.html` 文件即可。

## 数据源配置

编辑 `collect_news.py` 文件中的 `DATA_SOURCES` 列表，添加更多数据源：

```python
DATA_SOURCES = [
    {
        'name': '数据源名称',
        'url': '数据源URL',
        'type': 'web'  # 或 'rss'
    }
]
```

## 数据更新

### 自动更新（推荐）
- 设置定时任务，每周自动运行 `collect_news.py`
- 脚本会自动收集新闻并更新 `data.js` 文件

### 手动更新
1. 运行 `python collect_news.py` 收集数据
2. 或直接在 `data.js` 文件中手动添加数据

## 数据结构

每条新闻包含以下字段：
- `company`: 公司名称
- `title`: 标题
- `content`: 内容
- `priority`: 优先级（'重点' 或 '次重点'）
- `categories`: 分类数组（如：['AI教育', '教研']）
- `date`: 日期（格式：YYYY-MM-DD）
- `source`: 来源链接

## 注意事项

1. **网络连接**：数据收集需要网络连接
2. **数据源**：某些网站可能有反爬虫机制，需要根据实际情况调整
3. **数据质量**：自动收集的数据可能需要人工审核和调整
4. **浏览器兼容**：建议使用现代浏览器（Chrome、Firefox、Safari、Edge）

## 故障排除

### 收集不到数据
1. 检查网络连接
2. 检查数据源URL是否可访问
3. 查看脚本错误信息
4. 可能需要调整网页解析逻辑（网站结构可能变化）

### 数据格式错误
1. 检查 `data.js` 文件格式
2. 确保JSON格式正确
3. 检查日期格式
